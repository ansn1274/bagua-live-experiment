# 周易資料集

`web/data/zhouyi.json` 是從中文維基文庫整理出的繁體中文《周易》經傳資料。

## 內容

- 六十四卦，依通行卦序編號 1–64。
- 每卦卦辭、六條爻辭，以及乾坤的用九／用六。
- 每卦彖傳。
- 每卦象傳，分成大象與逐爻小象。
- 乾坤文言傳，保留原頁面的段落群組。
- 其餘十翼：繫辭上、繫辭下、說卦、序卦、雜卦，按章保存。
- 十翼索引明確分成彖上、彖下、象上、象下、文言、繫辭上、繫辭下、說卦、序卦、雜卦。

## 主要 Schema

```text
hexagrams[]
  number / name / aliases / glyph / composition
  classic
    judgment
    lines[6]
    specialLine
  wings
    tuan
    xiang.greatImage
    xiang.lineImages
    wenyan
  source

tenWings
  tuanUpper / tuanLower
  xiangUpper / xiangLower
  wenyan
  xiciUpper / xiciLower
  shuogua / xugua / zagua
```

每一卦及每一篇獨立傳文都保存維基文庫頁面與來源 URL。MediaWiki API 模式另保存 revision ID；browser bridge 快照則標記其擷取方式與時間。

## 更新

在 `web` 目錄執行：

```powershell
npm run data:zhouyi
```

若要從已擷取的 MCP Router browser bridge 快照重建：

```powershell
npm run data:zhouyi:browser
```

抓取器會重新下載資料並驗證：

- 恰好 64 卦與 384 條一般爻辭。
- 只有乾、坤有特殊用九／用六與文言傳。
- 每卦都有卦辭、彖傳、大象及至少六條小象。
- 五篇獨立十翼都有章節。
- 輸出中沒有殘留主要 Wiki markup。

來源：[中文維基文庫《周易》](https://zh.wikisource.org/wiki/周易)。

## Markdown 匯出

執行：

```powershell
npm run data:zhouyi:md
```

程式會在 `data/zhouyi-markdown/hexagrams/` 產生 `01-乾.md` 至 `64-未濟.md`，並在上一層建立 `README.md` 總索引。每個檔案包含 YAML metadata、卦辭、爻辭、彖傳、大象、小象、文言傳（若有）及來源資訊。

不屬於單一卦的繫辭上、繫辭下、說卦、序卦、雜卦會另外輸出到 `data/zhouyi-markdown/ten-wings/`，避免 Markdown 版本遺漏完整十翼內容。

要保留所有單檔並額外合併成一份 `data/zhouyi-markdown/周易全文.md`，執行：

```powershell
npm run data:zhouyi:combine
```

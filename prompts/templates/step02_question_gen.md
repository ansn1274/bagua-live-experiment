## User Template

不用貼給 LLM。此步驟由 `tools/prompt_generator.py` 自動寫入 `question.txt`。

## Expected Output Schema

```text
單一句問題文字。
```

## Usage Notes

- 類別清單需與標準解卦書一致。
- 產出會保存到 `data/cases/case_XXX/question.txt`。

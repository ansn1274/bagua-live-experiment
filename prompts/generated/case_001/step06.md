## System Prompt

你是一個故事語義判斷器，用來評估故事情境可能對應的梅花易數體卦與用卦。

你不是在起卦，也不是在猜數字。你只根據故事、問題與實際後續，分別估計八個卦作為「體卦」與「用卦」的可能性。

此任務的目標是把故事語義轉換成「體／用狀態分佈」，用來檢驗後續直覺取數產生的卦象，是否在語義上比隨機卦更貼近故事。

**請同時考慮前文中的未定狀態與後文中實際呈現出的發展。** 後半段的結果應該會影響你對體用狀態的判斷——當你看到事情實際的走向時，可能會修正或確認原先的印象。

### 體與用的操作定義

- **體卦（Body）**：故事中最接近「問事者」的位置。通常是主角、主要承受者、做決策的人，或問題句中真正關心其命運與狀態的一方。判斷體卦時，請看他的心理狀態、主動／被動程度、能量強弱、困境、等待、衝動、承受、表達、退讓等。
- **用卦（Use）**：故事中作用於體的外部力量。它可以是某個人、關係對象、團隊、制度、環境、事件壓力、疾病、考試、金錢流動、法律程序等。判斷用卦時，請看它如何影響體：是推動、阻隔、吸引、壓迫、承載、揭露、溝通、風險、變動，還是消耗。
- 若故事有多個角色，請以「問題真正問誰的處境」決定體；其餘對他產生主要作用者視為用。
- 若用不是單一人物，而是整個局勢或環境，請把「局勢的運作方式」當作用卦來評估。
- 不要把體卦理解為好壞結果，也不要把用卦理解為敵人。體／用只是主體與作用力的語義位置。

### 判斷流程

1. 先辨認故事的主要問事主體：誰正在承受不確定、需要作決策、或最受結果影響？
2. 再辨認主要外部作用力：是人、制度、疾病、資金、關係氣氛、考試規則、法律程序，還是環境變動？
3. 對體卦評估：主體在前半段與後半段中呈現的主導性、受困程度、等待、衝動、承載、溝通或退讓。
4. 對用卦評估：外部力量在實際後續中如何發揮作用，是推動、阻隔、揭露、風險、承載、耗損還是轉圜。
5. 最後將判斷轉成兩組八卦機率分佈。每組都可同時有 2～3 個高分卦，因為故事語義可能混合。

### 八卦狀態原型

請根據以下八卦的核心意象來判斷體與用各自的狀態分佈：

| 卦名 | 代號 | 核心意象與關鍵詞 |
|------|------|------------------|
| **乾** | qian | 主導、權威、強勢推進、決斷、上位者、制度壓力、企圖掌控局面。體為乾時常見於主角想主動突破；用為乾時常見於主管、規則、權威或強勢對象。 |
| **兌** | dui | 溝通、說服、交換、喜悅、社交、表態、口舌。體為兌時常見於主角依賴談判與表達；用為兌時常見於人際話語、邀約、承諾、輿論。 |
| **離** | li | 顯露、看見、依附、名聲、熱度、文件證據、表面清楚但內在牽連。體為離時常見於主角被看見或急於澄清；用為離時常見於曝光、評價、資訊揭露。 |
| **震** | zhen | 突發、驚動、啟動、衝擊、行動、變局。體為震時常見於主角被迫快速反應；用為震時常見於突發事件、消息、衝突或啟動壓力。 |
| **巽** | xun | 滲透、試探、反覆、順勢、猶豫、細微影響、逐步推進。體為巽時常見於主角不敢決斷、想慢慢觀察；用為巽時常見於曖昧風向、暗中影響、慢性變化。 |
| **坎** | kan | 風險、困境、陷落、恐懼、隱憂、疾病、債務、法律或心理壓力。體為坎時常見於主角焦慮受困；用為坎時常見於危機、陷阱、不可控風險。 |
| **艮** | gen | 停止、阻隔、等待、邊界、保守、卡住、靜觀。體為艮時常見於主角暫停、忍耐、守住底線；用為艮時常見於阻礙、門檻、拒絕、延遲。 |
| **坤** | kun | 承載、包容、配合、被動、照顧、資源基底、順承。體為坤時常見於主角承受、配合、照顧他人；用為坤時常見於家庭、群體、土地、背景資源或被動條件。 |

### 評分規則

1. 體卦與用卦各自輸出 8 個機率值，每組總和必須為 **1.0**。
2. 不要平均分配。最符合的卦通常應落在 0.25～0.45；次要卦可落在 0.12～0.25；弱相關卦應低於 0.10。
3. 若故事非常混合，可以分散一些，但仍需呈現明顯排序。
4. 體卦與用卦可以相同，也可以不同；請依故事語義判斷，不要刻意避開相同。
5. 不要使用梅花易數取數規則、卦序、數字、時間、方位來推斷。這一步只做語義評估。
6. 請同時考慮前半段的未定狀態與後半段的實際發展；若後半段揭露了外部力量的真實作用，應反映在用卦分佈中。
7. `brief_reason` 請一句話同時說明體與用的判斷依據，50 字以內。
8. 只輸出 JSON，不要加任何說明文字。

## User Message

故事前半段：<<<林曉晴把那杯已經涼掉的拿鐵推到一旁，眼睛還是盯著手機螢幕。陳宇的訊息停在昨晚十一點四十二分，只有短短一句：「我覺得我們需要談談。」她回了三個字，他沒有已讀。

她和陳宇認識七年了，從大學社團的同組夥伴，到後來幾乎每週都會見面的老朋友。她一直以為自己很清楚這段關係的輪廓——直到三個月前，她開始發現自己等他訊息的方式，和等其他人的不一樣。

咖啡廳的門被推開，帶進一陣六月的悶熱。

坐在她斜對面的男人抬起頭，兩人的視線意外交疊了一秒。她認不出他，但他的表情有一瞬間像是要開口說什麼，最後卻只是微微點了頭，低下頭繼續看桌上攤開的筆記本。曉晴移開視線，心裡有些奇怪的不安。

就在這時，手機震動了。

不是陳宇。是她最好的朋友方盈傳來的語音訊息，長達兩分鐘。她猶豫了一下，塞上耳機按下播放——方盈的聲音有些急，說昨晚陳宇打電話給她，問了一個讓方盈「不知道該不該告訴你」的問題。

語音在這裡停頓了將近三秒。

曉晴的手指懸在螢幕上，沒有動。斜對面那個陌生男人不知道什麼時候已經收起筆記本，正準備起身離開，他的椅腳在地板上劃出一道輕響。她沒有注意到。她只是靜靜地等著方盈說出下一句話，胸口有什麼東西正在慢慢收緊。>>>

問題：<<<關於這段人際互動，接下來整體狀態會如何發展？>>>

故事後半段：<<<方盈說：「他問我，你有沒有喜歡過他。」

語音就這樣結束了。

曉晴把耳機從耳朵裡拔出來，放在桌上。咖啡廳的背景音樂還在播，是某首她叫不出名字的英文歌，輕飄飄的，像是跟這個瞬間毫無關係。她看著螢幕上方盈的頭像，想回撥，手卻沒有動。

她不是不知道答案。

她只是還沒準備好承認，那個答案已經在她心裡住了多久。

窗外六月的街道曬得發白，偶爾有人騎著機車經過，帶起一點熱風。她想起三個月前某個週五的傍晚，陳宇在她家樓下等她，說要一起去吃她一直想試的那家越南河粉。他們走了很長一段路，他走在她左邊，偶爾替她擋住路邊停著的車。她當時沒有多想，現在卻覺得那個畫面清晰得有點刺眼。

手機再次震動。

這次是陳宇。不是訊息，是電話。

螢幕亮著，她看著他的名字一直閃，閃了大概五秒，然後停了。她沒有接。不是因為不想，而是因為她突然意識到，如果她現在接起來，她不確定自己說出口的會是什麼。

她重新拿起那杯涼掉的拿鐵，喝了一口，苦的。

過了一會兒，陳宇傳來一則訊息：「我在你家樓下附近。你在哪？」

曉晴盯著這句話看了很久。窗外的光把她的影子壓得很短，咖啡廳裡有人在笑，聲音隔著幾張桌子傳過來，顯得很遠。她深吸一口氣，拿起外套，把那杯沒喝完的咖啡留在桌上，往門口走去。

她還不知道見了面要說什麼。但她知道她要去。>>>
